About Us

This is a simple Online Note Categorization system where notes by users are automatically classified into different categories.

Web application is built by using Django framework, Python language, SQLite database and Html, CSS and Bootstrap for design.

Machine learning model is built by using 20newsgroups dataset, sklearn library and Multinomial Naive Bayes algorithm.
